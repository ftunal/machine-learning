# 🧠 Perceptron

Implementation of the Perceptron learning algorithm developed during the Artificial Intelligence course at PJATK.

## Features

- Binary Classification
- Perceptron Learning Algorithm
- Weight Adjustment
- Decision Boundary
- CSV Dataset Processing

## Dataset

- perceptron.data.csv
- perceptron.test.data.csv

## Technologies

- Java
- Machine Learning
- Artificial Intelligence
- Object-Oriented Programming

## How to Run

Run the `Main.java` file to train and evaluate the perceptron model.
